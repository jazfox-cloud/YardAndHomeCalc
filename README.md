# Yard & Home Calc

Free calculators for yard and home projects.

## Pages

- `/`
- `/mulch-calculator`
- `/concrete-calculator`
- `/paint-calculator`
- `/about`
- `/contact`
- `/privacy`
- `/terms`
- `/disclaimer`

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

The site is configured for static export. Production files are generated in `out/`.

## Cloudflare Pages

Recommended settings:

- Framework preset: Next.js static export or None
- Build command: `pnpm build`
- Build output directory: `out`
- Node.js version: 24 or compatible with Next.js 16
